一、BERT 模型
原理
基于 Transformer 双向编码器的预训练语言模型，通过掩码语言模型、下一句预测等任务学习通用上下文语义表示；下游任务在预训练权重基础上微调，接入全连接层与 Softmax 输出类别概率，属于深度语义理解型方法。
优点
具备强大上下文建模能力，可有效处理一词多义、歧义句、倒装、省略等复杂语言现象，分类准确率在标注数据充足时显著领先。
对低频词、小众类别、长尾样本的表征与泛化能力更强，覆盖更全面。
对口语化表达、错别字、语序混乱等文本噪声具备较强鲁棒性。
支持批量推理，设置合理 batch_size 可显著提升高并发、大批量场景的处理效率。
输出稠密低维语义向量，特征质量远高于传统统计方法，适配各类下游任务迁移。
缺点
开发与调参成本高，需要精细调节学习率、batch size、epoch、最大长度、学习率调度等超参数。
推理与训练对硬件要求高，依赖 GPU 或大内存服务器，CPU 环境速度缓慢。
依赖 HuggingFace Transformers 等框架，版本升级可能带来接口、权重兼容风险。
对输入长度敏感，标准模型最大长度受限，超长文本必须截断，易丢失关键信息导致误判。
模型权重文件体积大，一旦损坏或加载异常，会直接导致推理失效。
高度依赖高质量标注数据，样本量不足、标注噪声大时，精度会快速下降。
模型结构复杂，可解释性差，难以追溯分类决策的具体依据。
二、TF‑IDF
原理
基于词频与逆文档频率的统计特征工程，将文本转化为高维稀疏权重向量，再配合逻辑回归、SVM、朴素贝叶斯等传统机器学习模型完成分类，属于词袋统计特征型方法。
优点
实现简单、开发与调试成本低，无需复杂训练流程与硬件环境。
推理速度极快，内存占用极低，适合 CPU 环境与高并发轻量服务。
训练结果稳定，可复现性强，不存在随机初始化带来的大幅波动。
部署极度轻量化，无框架依赖，可直接打包上线，维护成本低。
对高频关键词区分明显的简单文本，精度显著优于规则正则，具备基础泛化能力。
缺点
本质是词袋模型，完全忽略语序、语法与上下文，无法处理多义词、歧义、隐含语义。
特征稀疏且维度高，在词汇量大时计算与存储开销上升，模型训练效率下降。
高度依赖分词效果与停用词表，分词错误、停用词更新不及时会直接影响特征质量。
泛化能力有限，对未登录词、全新句式、口语化表达容易分类错误。
对长文本、稀疏文本、低频词敏感，难以捕捉复杂语义关联。
无法理解语义相似度，近义词、同义表达无法统一表征。
三、LLM 大语言模型
原理
基于超大规模参数量的 Transformer 架构，通过海量无标注文本预训练获得通用语言理解、推理与生成能力；分类任务通过精心设计的 Prompt 指令 + 少量示例引导模型输出指定类别，属于指令遵循与零样本理解型方法。
优点
零样本 / 少样本条件下表现突出，精度远高于 TF‑IDF 与正则，部分场景可接近微调后 BERT。
无需训练、微调与大量标注数据，开发与迭代成本最低，上线速度快。
理解能力强，可处理复杂句式、隐含意图、隐喻、口语化表达等传统模型难以覆盖的场景。
新增或修改类别只需更新提示词，无需改动代码与模型结构，迭代灵活。
无框架版本兼容问题，通过 API 调用即可使用，部署维护简单。
支持上下文学习，搭配相似样例提示后，可进一步提升分类稳定性与准确率。
缺点
存在幻觉问题，可能输出不在预设类别内的结果，或给出看似合理但错误的分类。
对提示词格式、措辞、示例质量高度敏感，微小改动可能导致分类结果波动。
不同大模型的指令遵循能力、鲁棒性差异巨大，效果不可控因素较多。
推理延迟高，不适合超高并发、低延迟要求的实时场景。
私有化部署成本极高，API 调用会产生持续费用，且存在数据隐私与安全风险。
可解释性极差，无法给出分类依据，难以进行问题定位与规则回溯。
四、正则表达式
原理
通过人工编写关键词、通配符、句式结构等匹配规则，对文本进行字符串级精确匹配，命中规则则映射到对应类别，属于硬规则匹配型方法。
优点
规则完全可控、可解释，命中规则的样本分类结果 100% 符合人工预期。
对包含高频核心关键词的标准句式，分类准确率极高、无误差。
推理速度极快，计算与内存开销几乎可以忽略。
并发能力极强，理论上无上限，适合超高 QPS 简单场景。
开发、调试、部署成本极低，不依赖任何机器学习框架。
结果稳定可复现，不存在随机性，便于问题排查。
缺点
仅支持字面匹配，完全无法处理隐含意图、同义替换、歧义、口语化、省略句式。
覆盖能力有限，规则难以穷尽所有表达，大量样本只能归入其他类。
新增类别、新增表达方式必须手动编写与维护规则，迭代成本随业务增长快速上升。
规则数量庞大时，匹配效率下降，且易出现规则冲突、优先级混乱，需人工逐一解决。
无任何泛化能力，只能处理规则覆盖内的样本，对未知表达完全失效。
长期维护成本高，规则库膨胀后可读性变差，难以管理与升级。
