DeepSeek‑V3 相对 V2 的核心改进：

1. 整体架构与规模（最大变化）：V3 用更大总参数、更稀疏激活、翻倍上下文、多7倍数据，做更强通用模型。

2. V2 MoE用 辅助损失（auxiliary loss）做专家负载均衡，V3 用 Auxiliary‑Loss‑Free Load Balancing（无辅助损失负载均衡），训练更稳、更容易收敛。

3. V2 用 MLA（Multi‑Head Latent Attention），但未极致优化，KV 缓存仍然是长文本瓶颈；V3 MLA 全面升级，KV 缓存进一步压缩，长文本能力 + 推理速度双提升。

4.  V2 使用标准单 token 预测， V3 用 Multi‑Token Prediction（多 token 预测），直接提升推理与代码能力。

5. V2 用 FP16/BF16，显存占用高、通信开销大，V3 用 FP8 混合精度训练 + 推理，计算更快、通信更少。

6. 性能对比：上下文更大，速度更快，效果更好。

一句话总结 V2 → V3

更大 MoE（671B/37B）+ 无辅助损失负载均衡 + MLA 极致压缩 KV + 多 token 预测 + FP8 混合精度 + 14.8T 数据 + 128K 上下文 = 更强、更快、更长、更省的通用大模型。
